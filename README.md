# 110V-Single-Phase-to-220V-Three-Phase-Pure-Sine-Inverter-1kW
This project implements a high-efficiency AC-DC-AC power conversion system that converts a 110V single-phase input into a 220V three-phase pure sine wave output, rated at 1kW.
